Welcome to Jayhawk+!
This page will help with navigating our application


View profile

    Select profile icon
    Profile page shows all personal information needed

Booking an appointment

    Select booking icon
    Fill out personal information
    When completed, confirm appointment
    A confirmation message will appear
    You will be emailed the booking information
    Click prompted button to return to homepage

Canceling an appointments

    Go to Calendar
    Hover over appointment you want to cancel
    Click the x/cross button to remove appointment
    Select to confirm cancellation
